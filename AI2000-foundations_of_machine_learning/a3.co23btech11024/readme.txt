1) To run the files, make sure to have the following libraries
numpy, pandas, matplotlib, os, time, random, sys

If you don't have the instable libraries, run the following command
pip install numpy pandas matplotlib

2) To run run_spam_experiment.py use following steps
    > Make sure you in the directory containing run_spam_experiment.py from the submitted file
    > python run_spam_experiment.py

3) To run capstone_showdown.ipynb, open the respective file and run each cell one after the other

4) Best Model: MLP with layers [784, 256], [256, 128], [128, 10] (For train_frac=0.7, val_frac=0.15, test_frac=0.15), where activation b/w each layer is ReLU(). For more details visit the report

5) To get computational graph for problem 4
    > Make sure you in the directory containing visualize.py from the submitted file
    > python visualize.py
